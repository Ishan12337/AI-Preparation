# 🚀 AI Interview Preparation Platform

A full-stack AI-powered web application that helps users prepare for interviews using smart resume analysis, ATS scoring, and AI-generated questions.

---

## 📂 Project Structure

project/
│
├── frontend/ # React Application (UI)
├── backend/ # Node.js + Express API
└── README.md



---

## 🌟 Features

- 🔐 User Authentication (Login / Register)
- 📄 Resume Upload (PDF support)
- 🤖 AI-generated Interview Questions
- 📊 Match Score Analysis
- 🧠 Technical & Behavioral Questions
- 📅 Personalized Preparation Plan
- 📁 Dashboard with saved reports

---

## 🛠️ Tech Stack

### Frontend
- React.js
- SCSS (Glassmorphism UI)
- Axios

### Backend
- Node.js
- Express.js
- MongoDB

## 🔄 Application Flow

1. User registers / logs in  
2. Uploads resume + job description  
3. Backend processes data  
4. AI generates interview questions  
5. User views report in dashboard
